# Multimodal Face & Voice Anti-Spoofing Detection System

A multimodal deep learning project focused on detecting fake or manipulated audio-video content using both facial and voice-based analysis.

This system combines computer vision and audio processing techniques to classify inputs into categories such as real video/audio, fake video, and spoofed audio combinations. The project was developed as a B.Tech final-year project and explores how multimodal learning can improve biometric security and deepfake detection.

## Features

* Audio-visual spoof detection using multimodal analysis
* Video frame extraction and preprocessing
* Mel spectrogram generation for audio analysis
* Deep learning models using ResNet18, CNN, and LSTM
* Flask-based web interface for real-time prediction
* User authentication and secure upload workflow
* Real-time inference and spoof detection alerts

## Tech Stack

* Python
* PyTorch
* Flask
* OpenCV
* Librosa
* NumPy & Pandas
* HTML, CSS, JavaScript

## Models Used

* ResNet18 for spatial feature extraction
* LSTM for temporal sequence modelling
* CNN-based audio feature extraction
* Feature-level fusion for multimodal classification

## Project Goal

The objective of this project is to improve face anti-spoofing systems by combining visual and auditory information instead of relying on a single modality. The system aims to detect inconsistencies between facial movements and voice signals in order to identify deepfake or manipulated media more effectively.
